## ğŸ¨ Customization

### Personalizing Your Portfolio

1. **Update Personal Information**
   - Edit `index.html` to include your name, title, and bio
   - Replace placeholder images in `assets/images/`
   - Update contact information in `contact.html`

2. **Modify Colors and Themes**
   ```css
   /* In style.css, update CSS variables */
   :root {
     --primary-color: #your-color;
     --secondary-color: #your-color;
     --accent-color: #your-color;
   }
   ```

3. **Add Your Projects**
   - Update the projects section with your own work
   - Include live demos and GitHub repository links
   - Add project screenshots and descriptions

4. **Social Media Links**
   - Update social media icons and links in the footer
   - Customize the contact form to use your preferred service
